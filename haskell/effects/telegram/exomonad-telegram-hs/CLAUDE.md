# exomonad-telegram-hs - Haskell Telegram Bot Integration

Haskell library for Telegram bot integration within the ExoMonad framework.

## What This Is

Provides Telegram bot effect types and integration code for Haskell-based ExoMonad agents.

## Status

**Check codebase** - This package may contain legacy or experimental Telegram integration code. The primary Telegram integration is in `exomonad-telegram-ts` (TypeScript).

## Usage

See `exomonad-core` for the core `ExoMonad.Effects.Telegram` module which defines the effect types used by agents.
