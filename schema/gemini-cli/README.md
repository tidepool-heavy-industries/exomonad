# Gemini CLI Settings Schema

This directory contains the JSON schema for verifying `settings.json` configuration files for the Gemini CLI.

## Source

- **URL:** `https://raw.githubusercontent.com/google-gemini/gemini-cli/refs/heads/main/schemas/settings.schema.json`
- **Downloaded:** 2026-02-11

## Usage

Use the `just validate-settings` command to verify your `.gemini/settings.json` against this schema.
