# tidepool-telegram-hs - Haskell Telegram Bot Integration

Haskell library for Telegram bot integration within the Tidepool framework.

## What This Is

Provides Telegram bot effect types and integration code for Haskell-based Tidepool agents.

## Status

**Check codebase** - This package may contain legacy or experimental Telegram integration code. The primary Telegram integration is in `tidepool-telegram-ts` (TypeScript).

## Usage

See `tidepool-core` for the core `Tidepool.Effects.Telegram` module which defines the effect types used by agents.
